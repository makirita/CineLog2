# 「CineLog」とは？
映画の感想を記録できるアプリです。  
TMDbのAPIを使い、映画を検索することができ、マイリストに登録。登録した映画のレビューが書けます。  
他の人のレビューを見ることはできず、自分のレビューが公開されることもない完全に自分だけの記録アプリとなっております。
